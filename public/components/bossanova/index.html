Accordion

Demonstration

Inline load option

Inline static HTML content

Remote load option

Methods

Method: open (smallint [tabNum]);

// Open the tab number. Tab index starting on 0
Example: $('#myaccordion').accordion('open', 1);



Method: add (string [title], string [html]);

// Add a new item with static HTML inside the accordion
$('#myaccordion').accordion('add', 'Your title', '<b>new item</b>');



Method: load (smallint [tabNum], string [html]);

// Load a static HTML inside an element in the accordion
Example: $('#myaccordion').accordion('load', 0, '<b>new item content</b>');



Method: remove (smallint [tabNum]);

// Remove an element from the accordion
Example: $('#myaccordion').accordion('remove', 0);



Method: addUrl (string [title], string [url]);

// Add a remote HTML content inside in a new element and append in the accordion
Example: $('#myaccordion').accordion('addUrl', 'title remote content', 'http://www.bossanova-framework.com/tests/test');



Method: loadUrl (string [title], string [url]);

// Load a static HTML inside an element in the accordion
Example: $('#myaccordion').accordion('loadUrl', 0, 'http://www.bossanova-framework.com/tests/test2');

Code example

<html> <head> <script src="https://code.jquery.com/jquery-2.1.4.min.js"></script> <script src="http://www.bossanova-framework.com/ui/js/jquery.accordion.js"></script> <link rel="stylesheet" href="http://www.bossanova-framework.com/ui/css/style.css" type="text/css" /> <link rel="stylesheet" href="http://www.bossanova-framework.com/ui/css/system/jquery.accordion.css" type="text/css" /> </head> <body> <div id="myaccordion" style="width:220px;height:300px;"> <div> <h1><span>Inline load option</span></h1> <div>Inline static HTML content</div> </div> <div> <h1><span>Remote load option</span></h1> <div url="http://www.bossanova-framework.com/tests/test"></div> </div> </div> </body> </html>
<script> $("#myaccordion").accordion(); $("#myaccordion").accordion('open', 0); // Open the first tab </script>

Calendar

Methods

// Create a calendar picker element based on existing input. If calendar already exists open the calendar picker.
// format: {String} calendar format. ex. 'dd/mm/YYYY';
// dynamicformat: {Method} any function to return the format live
// readonly: {Boolean} input text is a readonly [0/1]
// clock: {Boolean} show the hour and minutes picker [0/1]
// months: {Array} array of string so can be translated
// weekdays: {Array} array of string so can be translated

Method: init (json [options]);

Example: $('#mycalendar').calendar();



// Open the calendar if the element exists
Method: open ();

Example: $('#mycalendar').calendar('open');



// Close calendar if it is openned
Method: close ();

Example: $('#mycalendar').calendar('close');



// Reset the current value of the calendaer
Method: reset ();

Example: $('#mycalendar').calendar('reset');

Code example

<html> <head> <script src="https://code.jquery.com/jquery-2.1.4.min.js"></script> <script src="http://www.bossanova-framework.com/ui/js/jquery.calendar.js"></script> <link rel="stylesheet" href="http://www.bossanova-framework.com/ui/css/style.css" type="text/css" /> <link rel="stylesheet" href="http://www.bossanova-framework.com/ui/css/system/jquery.calendar.css" type="text/css"/> </head> <body> <input type="text" id="mycalendar" name="mycalendar"/> </body> </html>
<script> $("#mycalendar").calendar(); </script>

Tabs

Methods

Method: add (json [options]);

// Add a new tab (if the index provied already exists display and focus the tab)
Example: $('#mytabs').tabs('add', {title:'New tab', index:'unique_name', html:'this is a new tab and content', closable:'1'});



Method: open (string [tabIndex]);

// Close an existing tab by index name
Example: $('#mytabs').tabs('open', 'second_tab');



Method: close (string [tabIndex]);

// Close an existing tab by index name
Example: $('#mytabs').tabs('close', 'second_tab');
